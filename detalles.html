Justificaciones

En este apartado encontrara la justificación de cada componente utilizado.

Input Nombre

Este input sirve para capturar el nombre del cliente en el formulario.

Input Contraseña

Este input sirve para capturar la contraseña del cliente en el formulario.

Input Repetir Contraseña

Este input sirve para confirmar la contraseña ingresada y hacer una validación en el formulario.

Input Direccion

Este input sirve para capturar la direccion del cliente en el formulario.

Input Ciudad

Este input sirve para capturar la ciudad del cliente en el formulario.

Input Numero de Huespedes

Este input sirve para capturar la provincia del cliente en el formulario.

Input Ciudad

Este input sirve para capturar la cantidad de huespedes en la habitación.

Servicios

Este apartado nos sirve para saber que servicios va a necesitar el cliente en su habitación.

Tipos de Habitación

Este apartado nos sirve para saber que tipo de habitación desea el cliente.

Pensión

Este apartado nos sirve para saber que servicios de pension necesita el cliente.

Comentarios o sugerencias

Este apartado nos sirve para que el cliente ingrese los comentarios, quejas o sugerencias que tenga de nuestros servicios

Pensión

Este apartado nos sirve para que el cliente nos deje su correo de contacto y así poder comunicarnos con el.

Pensión

Este apartado nos sirve para informarle al cliente sobre las politicas de la empresa y lo que haremos con su información, y de esta forma evitarnos inconvenientes.

Boton Enviar

Este apartado nos sirve para enviar toda la información del formulario para validarla y enviarla a nuestro Back-end.

Boton Reset

Este apartado nos sirve para borrar la información que ya había escrito el cliente en los inputs, checkbox y textarea.
